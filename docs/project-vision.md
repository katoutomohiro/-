# ピノキオ重心ケアシステム - プロジェクトビジョン

## 事業所概要

### 重症心身障がい児者多機能型事業所「ピノキオ」
- **事業種別**: 重症心身障がい児者多機能型事業所
- **サービス内容**: 
  - 生活介護事業（成人向け）
  - 放課後等デイサービス事業（児童向け）
- **利用者数**: 計24名
  - 生活介護: 14名（成人・重症心身障がい者）
  - 放課後等デイサービス: 10名（児童・発達障がい等）

### 利用者構成

#### 生活介護（14名）
- 年齢層: 41-60歳
- 主な障害: 脳性麻痺
- 医療的ケア: 吸引、経管栄養、腸瘻など高度な医療的ケアが必要
- 介護度: 要介護4-5が中心

#### 放課後等デイサービス（10名）
- 年齢層: 9-18歳
- 主な障害: 自閉症スペクトラム、ダウン症、脳性麻痺、ADHD、知的障害
- 支援度: 支援度1-6
- 医療的ケア: 一部利用者に吸引等が必要

## プロジェクトの最終目的

### 職員の業務負担6割削減の実現
- **現状課題**: 
  - 紙ベースの記録作業による時間的負担
  - 情報共有の非効率性
  - モニタリングデータの分析困難
  - 個別支援計画作成の負担

- **目標**: 
  - デジタル化による記録時間の大幅短縮
  - リアルタイム情報共有の実現
  - データ分析による科学的根拠に基づく支援
  - 職員がより直接的な支援に集中できる環境の構築

## AI エージェントの役割分担（統合）

このプロジェクトでは複数の AI エージェントが協働して開発を進めます。以下は統合された役割分担と推奨ワークフローです。

### エージェント一覧
- v0（戦略的設計者・アーキテクト）
  - 全体設計、機能要件の定義、設計書とコードの生成を行います。
  - 大規模な設計変更や複数ファイルにまたがる実装は v0 の設計に従って進めます。
- Codex CLI（実行担当）
  - リポジトリ内のファイル操作、コマンド実行（pnpm/tsc/git 等）、CI チェック、Git 操作を実行します。
  - v0 が生成した変更をファイルに書き込み、テスト・型チェックの実行、コミット・プッシュまでを担当します。
- VS Code 内エージェント（インライン支援）
  - IDE 内での小規模修正、コード補完、ドキュメントコメントの追加、現場向けの細かな最適化を担当します。

### 推奨ワークフロー（統合）
- 新機能開発:
  1. v0 が設計と初期コードを生成。
  2. Codex CLI が生成結果をリポジトリに書き込み（大規模変更）、pnpm tsc 等でエラーチェックを実行。
  3. VS Code 内エージェントが微調整とコメント追加を行う。
  4. Codex CLI が最終の git commit & push を実行。
- バグ修正（小規模）:
  - VS Code 内エージェントが即時修正 → Codex CLI がエラーチェックと Git 操作。
- バグ修正（大規模）:
  - v0 が修正計画を提示 → Codex CLI が編集とチェック → VS Code 内エージェントが調整 → Codex CLI が Git 操作。

### 衝突回避ルール
- Codex CLI が大規模な変更を書く前に、VS Code 内エージェントの作業を保存してから進める。
- VS Code 内エージェントが編集中の場合は書き込みを待機する。
- Git 操作は原則 Codex CLI が実行する（自動的なリモート操作はチーム合意のもとで実行）。
